MyPetPrints – Artwork Approval Automation System

An automated workflow system that connects Shopify orders with a custom admin web app, Trello, Pabbly workflows, webhooks, and cron jobs to manage artwork upload, approval, revisions, reminders, and escalation rules.

Shopify PHP MySQL Trello API Webhooks Pabbly Cron Jobs Email Automation
View GitHub Repo Live Store

What I Built

  • Shopify order webhook → inserts order into admin web app and triggers Trello card creation.
  • Automated Trello list movement based on customer actions (approve / revise / no-reply).
  • Customer portal flow: login using order number + email, approve artwork, request revisions, select background.
  • Reminder + escalation logic: repeat emails at intervals, auto-approve after time window.

Automation Logic

  • Cron jobs run every ~5 minutes to fetch cards and sync artwork status.
  • If artwork is missing: auto-upload + send customer approval email.
  • If customer doesn’t respond within the defined window: auto-approve and/or move to No Reply.
  • Revision requests add comments to Trello + move card into Revision Request list.

Tech Stack

  • Backend: PHP + MySQL (custom admin web app)
  • Integrations: Shopify Webhooks, Trello API
  • Automation: Pabbly workflows + cron jobs
  • Notifications: automated email reminders

Highlights / Outcomes

  • Reduced manual effort by automating card creation, file sync, and follow-ups.
  • Clear approval pipeline: Pending → Approved / Revised / Rejected / No Reply.
  • Scalable workflow design for multi-order processing.
