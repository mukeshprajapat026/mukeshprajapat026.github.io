Project Overview

The client was manually managing artwork approvals via email and Trello, causing delays and operational inefficiencies. I built a fully automated workflow system connecting Shopify orders with a custom admin panel, Trello boards, and scheduled automation processes.

What I Built
  • Shopify webhook → auto-create admin records
  • Trello card automation & list movement
  • Customer approval portal (order + email login)
  • Reminder & escalation logic system
Automation Logic
  • Cron jobs every 5 minutes
  • Auto email approvals
  • No-response → escalation workflow
  • Revision sync with Trello API
Business Impact
  • Reduced manual workload by ~60%
  • Improved artwork approval turnaround time
  • Automated 100+ order workflows
  • Scalable multi-order processing
Tech Stack
  • Backend: PHP + MySQL
  • Shopify Webhooks
  • Trello API
  • Cron Jobs & Email Automation
